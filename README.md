# Backpack Vue PWA

A Vue.js Progressive Web Application (PWA) for Backpack

## Requirements
* Node.js
* Vue CLI

## Installation
```
git clone https://github.com/backpack-io/vue-pwa.git
cd vue-pwa
npm install
```

## Run
```
npm run serve
```
